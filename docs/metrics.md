# Quality metrics and evaluation

## Retrieval quality

- Precision@K: fraction of top K papers judged relevant.
- Coverage: number of unique sources represented in top K.
- Freshness: share of papers published today.

## Ranking quality

- NDCG@K: ranking quality vs. labeled relevance.
- Query match rate: percentage of papers with strong keyword overlap.

## Output quality

- Readability: average sentence length and bullet count.
- Summary coverage: average abstract overlap for summaries.

## Evaluation plan

- Collect a small labeled dataset for 3-5 queries.
- Run weekly evaluation and track metrics in a CSV.
- Spot check the digest for clarity and novelty.
